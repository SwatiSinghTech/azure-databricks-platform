# Databricks SAT — Security Analysis Tool Setup

## What is SAT
Databricks Security Analysis Tool scans workspace
and gives security posture score + recommendations.

## What SAT Checks
- Network security configuration
- Workspace access controls
- Unity Catalog enforcement
- Cluster security settings
- Secret management

## Prerequisites
- Python 3.8+
- Databricks CLI configured
- Workspace Admin access
- PAT token
